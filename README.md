## 🎯 KEY FEATURES

- **Adaptive Gameplay**: Captain responds to boundaries, not singles
- **Dumb Mistakes**: Low-rated players play wrong shots
- **Player Dynamics**: Confidence, fatigue, pressure affect performance
- **Realistic Venues**: Each stadium has unique characteristics
- **4 Commentary Styles**: Choose your viewing experience
- **Full Scorekeeping**: Every run, wicket, ball tracked
